COMMAND: ON
===========
**Available triggers:**
- hurt
- smelt
- craft
- toss
- heal
- teleport
- starve
- entityjoin
- achievement
- hungerregen (requires AppleCore)
- item usage

**Available reactions:**
- explode
- damage
- applyeffect
- addexperience
- cancel
- sprinting
- invisibility
- forcing difficulty

**Available arguments:**
- minamount (decimal)
- maxamount (decimal)
- minhealth (decimal)
- maxhealth (decimal)
- item (item)
- minarmor (integer)
- maxarmor (integer)
- damagetype (string)
- minchance (integer)
- maxchance (integer)
- entitytype (string)
